PMP A4P reconstruit — compatible hub

Contenu du pack :
- index.html
- styles.css
- script.js
- questions.json
- mapping.json

Ce pack est opérationnel dans GitHub Pages.
Il enregistre aussi ces clés pour la synthèse du hub :
- a4p_pmp_profile
- a4p_pmp_mbti
- a4p_pmp_action_code
- a4p_pmp_scores
- a4p_pmp_completed

Déploiement :
1. Ouvre le dépôt GitHub du PMP
2. Supprime les anciens fichiers index/script/style si besoin
3. Envoie tout le contenu de ce pack à la racine du dépôt
4. Attends quelques secondes puis recharge le lien GitHub Pages
